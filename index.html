기술 스택

주요 핵심 기술

LoRA 파인튜닝

대규모 언어 모델(LLM) 또는 비전 모델을 특정 작업이나 도메인에 맞게 효율적으로 미세 조정하는 기술입니다. 적은 파라미터만 업데이트하여 계산 비용과 시간을 절약하면서 높은 성능을 달성합니다.

RAG (Retrieval Augmented Generation)

외부 지식 베이스에서 관련 정보를 검색하여 대규모 언어 모델의 답변 생성을 보강하는 기술입니다. 모델이 최신 정보를 반영하고 환각(hallucination)을 줄이는 데 효과적입니다.

머신러닝 기본 원리 이해

지도 학습, 비지도 학습, 강화 학습 등 다양한 머신러닝 알고리즘의 수학적 원리와 작동 방식에 대한 깊이 있는 이해를 바탕으로 모델 선택, 성능 평가, 문제 해결 능력을 갖추었습니다.

상태 추정 알고리즘 (EKF, DEKF)

확장 칼만 필터(EKF) 및 이중 확장 칼만 필터(DEKF)와 같은 고급 상태 추정 알고리즘 활용 경험이 있습니다. BMS 연구에서 배터리의 SOC, SOH를 실시간으로 정밀하게 추정하여 시스템 효율성과 안정성을 최적화했습니다.

TTS 시스템 구현

자연스럽고 표현력 있는 음성을 합성하는 Text-to-Speech 시스템 설계 및 구현 경험이 있습니다. 음성 데이터 전처리, 모델 학습, API 개발을 포함하여 사용자 맞춤형 음성 서비스를 제공할 수 있습니다.

백엔드 개발 (FastAPI)

Python 기반의 FastAPI를 활용한 고성능 백엔드 API 개발 경험이 있습니다. 비동기 처리, 데이터 유효성 검사 등을 통해 안정적이고 확장 가능한 서비스를 구축합니다.

백엔드 개발 (SpringBoot)

Java 기반의 SpringBoot 프레임워크를 활용한 엔터프라이즈급 백엔드 시스템 개발 경험이 있습니다. RESTful API 설계, 데이터베이스 연동, 보안 기능 구현 등에 익숙합니다.

프로그래밍 언어

Python (주력)

데이터 분석, 머신러닝/딥러닝 모델 개발, 웹 백엔드 구축 등 AI 개발 전반에 능숙하게 활용합니다. PyTorch, TensorFlow 등 다양한 라이브러리 사용 경험이 풍부합니다.

C (with STM32)

STM32 마이크로컨트롤러를 활용한 펌웨어 개발 및 임베디드 시스템 제어 경험이 있습니다. 하드웨어 직접 제어 및 최적화가 중요한 BMS 연구 등에서 활용했습니다.

MATLAB

BMS 제어 연구 시 알고리즘 개발, 시뮬레이션 및 데이터 분석에 활용하였으며, 제어 시스템 설계 및 검증 경험을 보유하고 있습니다.

딥러닝/AI 프레임워크

PyTorch

유연성과 Python 친화성을 바탕으로 다양한 딥러닝 모델(CNN, RNN, Transformers 등) 연구 및 개발을 수행합니다. 동적 계산 그래프를 활용한 효율적인 모델 구축이 가능합니다.

TensorFlow

Keras를 포함한 TensorFlow 생태계를 활용한 딥러닝 모델 개발 및 배포 경험이 있습니다. 대규모 데이터 처리 및 분산 학습 환경에서의 모델링에 익숙합니다.

Hugging Face Transformers

최신 NLP 모델 및 트랜스포머 아키텍처를 활용합니다. 사전 학습 모델 파인튜닝, 텍스트 생성, 감성 분석 등 다양한 NLP 작업 수행이 가능합니다.

Langchain

LLM을 활용한 애플리케이션 개발 프레임워크로, RAG, 에이전트, 체인 구성 등을 통해 복잡한 LLM 기반 시스템 구축 경험이 있습니다.

Sesame, Zonos (TTS 관련 확인 필요)

Sesame TTS는 감정 표현이 풍부한 음성 합성에, Zonos는 [Zonos 관련 내용 추가 필요]에 활용되었습니다. 프로젝트에서 자연스러운 원어민 음성 제공에 기여했습니다.

기타 도구 및 라이브러리

Git

분산 버전 관리 시스템을 사용하여 코드 변경 사항을 효율적으로 추적하고 협업 프로젝트를 관리합니다. 브랜칭, 병합 등 워크플로우에 대한 이해가 있습니다.

Docker

컨테이너화 기술을 활용하여 개발 환경을 표준화하고 애플리케이션을 격리하여 배포 일관성을 확보합니다. Docker 이미지 생성 및 컨테이너 관리 경험이 있습니다.

Scikit-learn

다양한 머신러닝 알고리즘과 모델 평가 도구를 제공하는 핵심 Python 라이브러리입니다. 기본적인 AI 개념을 정리하고 팀 내 공유 및 스터디 자료로 활용하여 AI 지식 전파에 기여했습니다.

NumPy

Python에서 과학 컴퓨팅을 위한 기본 패키지입니다. 다차원 배열, 선형 대수, 푸리에 변환 등을 제공하여 머신러닝 및 데이터 분석의 기반 기술로 활용됩니다.

Simulink

MATLAB 기반의 블록 다이어그램 환경으로, 동적 시스템 및 임베디드 시스템의 모델링, 시뮬레이션 및 분석에 사용됩니다. BMS 제어 로직 설계 및 검증 경험이 있습니다.

프로젝트

오마이픽 (Omypic)

프로젝트 설명:

‘오마이픽'은 사용자의 경험을 기반으로 개인 맞춤형 OPIc 스크립트 생성을 돕고, 실제 시험과 유사한 환경에서의 연습과 AI 기반 상세 피드백을 통해 OPIc 학습 효과를 극대화하는 서비스입니다. 2030 취업준비생들의 주된 어려움인 스크립트 작성, 실전 연습 부족, 피드백 부재를 해결하고자 기획되었습니다.

역할 및 기여 (Back/AI):
  • FastAPI 기반 백엔드 시스템 설계 및 API 개발 참여
  • AI 모델 연동 및 파이프라인 구축 (STT, LLM, TTS)
  • Celery와 Redis를 활용한 비동기 작업 처리 시스템 구현 (사용자 답변 평가 로직 등)으로 서버 부하 감소 및 응답 속도 개선
  • Gemini를 활용한 사용자 답변 평가, 예상 등급 산정 및 문법/어휘/발화량에 대한 세부 피드백 생성 로직 개발
  • Emotion BERT Routing 시스템 (Sesame, gTTS, BERT) 도입을 통한 TTS 음성 생성 시간 83% 단축 및 자연스러운 원어민 음성 품질 확보
  • AI 프롬프트 엔지니어링을 통한 스크립트 길이 및 피드백/예상 등급 정교화
  • 데이터베이스(MongoDB) 스키마 설계 및 연동 참여
사용 기술:

AI/ML: Python, PyTorch, Hugging Face Transformers, LangChain, Gemini, Whisper, Sesame, gTTS, BERT (Emotion BERT Routing)
Backend: Python FastAPI, Celery, Redis, MongoDB
Infra: Docker, Jenkins, Nginx, AWS EC2, S3, Prometheus, Grafana, Flower
Frontend (팀 기술): React, TypeScript, Vite, Redux, Tanstack Query, PWA

주요 기능 및 구현 내용:
  • AI 기반 스크립트 생성: 사용자와의 한국어 대화를 통해 개인 경험 기반의 영어 OPIc 스크립트 자동 생성 (Gemini 활용)
  • 원어민 발음 TTS: 생성된 스크립트를 자연스러운 원어민 발음으로 청취 (Emotion BERT Routing: Sesame, gTTS, BERT)
  • 실전 모의고사: 실제 OPIc 시험과 유사한 UI/UX 환경에서 Background Survey 선택, 문제 청취, 답변 녹음 기능 제공
  • AI 상세 피드백: 녹음된 답변에 대해 STT(Whisper) 변환 후 AI(Gemini)가 문법, 어휘, 발화량, 유창성 등을 분석하여 예상 등급 및 구체적인 개선 방안 제시
  • 유형별 연습: 맛보기, 콤보셋, 롤플레잉, 돌발문제 등 OPIc 주요 문제 유형별 집중 연습 기능
  • 비동기 처리: Celery, Redis를 도입하여 AI 평가 등 시간이 소요되는 작업을 비동기적으로 처리하여 사용자 경험 향상

MCPanda - Agentic AI 학습 환경

프로젝트 설명:

LLM의 수학적 추론 능력 향상과 Agentic AI 기반 학습 환경 구축을 목표로 한 프로젝트입니다. Supervised Fine-Tuning 데이터셋 구축과 자동화된 환경 설정을 통해 AI 학습 효율성을 극대화했습니다.

역할 및 기여:
  • Supervised Fine-Tuning 데이터셋 설계 및 구축
  • Agentic AI를 활용한 자동화된 학습 환경 설계
  • Perplexity 모니터링을 통한 학습 안정화 (Early Stopping, Dropout, Learning Rate 조정)
  • 제한된 VRAM 환경(10GB)에서 Unsloth 4bit Q_LoRA 기법 적용으로 메모리 효율성 향상
  • Hidden Math 지표를 활용한 모델 추론 능력 정량적 평가
사용 기술:

AI/ML: Python, PyTorch, Hugging Face Transformers, LoRA, Q_LoRA (Unsloth), RAG
기타: Git, Agentic AI Framework

주요 구현 내용:
  • Supervised Fine-Tuning 데이터셋: 수학 추론 성능 향상을 위한 고품질 학습 데이터 구축
  • Agentic AI 환경: 자동화된 학습 환경 설정 및 관리 시스템 구현
  • 효율적 파인튜닝: 4bit Q_LoRA 기법으로 메모리 제약 극복
  • 성능 모니터링: Perplexity 추적 및 학습 과정 최적화
성과:

제한된 하드웨어 환경에서 대규모 언어 모델의 효율적 파인튜닝 방법론 확립. Agentic AI를 통한 자동화된 학습 환경으로 개발 생산성 향상. Q_LoRA 경량화 기법 실제 적용 경험 확보.
